
Q1: Why might we consider using Git?

Answer: It's great for collaborating on code for group projects and easy to use for adding many different files to a coding project.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science could be applied to any field, including finance. Data science could be used to keep track of stock prices and predict what will come in the future.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging applies the changes made in a certain branch to the current branch. Rebasing is when the branches are put on top of each other and it replays the changes made from the feature branch to the current branch.
